Tic-Tac-Toe AI

Project Overview:
The Tic-Tac-Toe AI project is a Python-based game where a human player competes against an AI opponent. The AI uses the Minimax Algorithm to analyze the game board and make the best possible move at each turn.
The objective of this project is to demonstrate how Artificial Intelligence can be applied to game development and decision-making. The AI evaluates different game outcomes and chooses moves that maximize its chances of winning while preventing the opponent from winning.
This project helps in understanding important AI concepts such as Game Theory, Search Algorithms, Recursion, and Decision Making.

## Features
- Human vs AI gameplay
- AI powered by Minimax Algorithm
- Win and Draw detection
- Input validation
- Console-based interface
- Unbeatable AI strategy

## Technologies Used

- Python
- Minimax Algorithm
- Recursion

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/CODSOFT.git
```

2. Navigate to the project folder:

```bash
cd Task2_TicTacToe_AI
```

3. Run the program:

```bash
python tic_tac_toe.py
```
