Rule-Based Chatbot
Project Overview:
This project is a Rule-Based Chatbot developed using Python.
The chatbot responds to user queries using predefined rules and if-else conditions. It demonstrates the fundamentals of Natural Language Processing (NLP) and conversational AI.
-----------------------------
Features:
- Greeting responses
- Date and time information
- Help menu
- About chatbot information
- Internship-related response
- Exit functionality
- Fallback responses for unknown inputs
-----------------------------
Technologies Used:
- Python 3
- Datetime Module
- Random Module
-----------------------------
How to Run:
Step 1
Clone the repository

```bash
git clone <repository_link>
```
Step 2
Navigate to project folder

```bash
cd Task1_Chatbot
```
Step 3
Run the chatbot

```bash
python Task1.py
```
--------------------------------
Sample Interaction
```text
You: hello
Bot: Hello! How can I help you?

You: time
Bot: Current time is 08:30:15 PM

You: what is your name
Bot: I am a Rule-Based Chatbot developed for the CodSoft AI Internship.

You: bye
Bot: Thank you for chatting. Goodbye!
```
