Movie Recommendation System
📌 Project Description

This project is a simple Movie Recommendation System developed using Python. It recommends similar movies based on their genres using the Content-Based Filtering technique.

The system converts movie genres into numerical vectors using CountVectorizer and calculates similarity between movies using Cosine Similarity. When a user enters a movie name, the system recommends the top similar movies.

🚀 Features
Recommend movies based on genre similarity.
Fast and simple recommendation system.
Uses Content-Based Filtering.
Beginner-friendly AI project.
